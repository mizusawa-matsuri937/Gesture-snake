# Gesture Snake Development Log

Date: 2026-06-03

## Duo Mode Foundation

`Duo Mode` now has a first shared-camera implementation. The flow is `Duo Mode` -> control setup -> shared-camera map select -> battle. `Separate Cameras` is visible as Coming Soon.

Shared-camera battles split the camera preview into left and right halves. The left half controls the green snake and the right half controls the blue snake. Both players must be detected in their own half for 0.5 seconds before the match starts; tracking loss or center-line crossing pauses the active timer.

The mode reuses the five endless challenge maps. Matches last 3 minutes of active play, use fixed x8 sensitivity, share normal and big apples, wrap map edges, and keep per-snake portal cooldowns. Walls, moving walls, self-collision, opponent body collision, and head-on collisions end the match. Snake death deducts 100 points, scores may go negative, and tied scores show Draw.

The screenshot harness now covers Duo control setup, map select, waiting, playing, paused, gameover, and separate-camera Coming Soon states.

## Boundary Wrap Fix

Classic endless play and all Endless Challenges now wrap by the snake head radius. When the head reaches a map edge, it appears from the opposite edge without creating a cross-map body trail. Level Mode still keeps boundary collision as a Game Over condition.

Date: 2026-06-02

## Current Architecture

The project is split into focused modules:

- `snake.py`: entry point.
- `game.py`: pygame setup, main loop, state machine, mode switching.
- `ui.py`: buttons, menu screens, overlays, sidebars, and entity drawing.
- `vision.py`: camera capture, MediaPipe hand tracking, gesture triggers.
- `entities.py`: snake, food, walls, moving walls, and portals.
- `modes/single_mode.py`: classic endless mode.
- `modes/endless_challenge_mode.py`: selectable endless challenge maps.
- `modes/level_mode.py`: target-based staged mode.
- `modes/obstacle_helpers.py`: shared obstacle, portal, layout validation, and safe food spawning helpers.

## Endless Challenge Level Select

`Single Player` now opens an `Endless Challenges` selection screen instead of immediately starting classic endless mode. The screen uses a two-column card layout with five challenges:

1. `Classic`
2. `Symmetry`
3. `Moving Walls`
4. `Portals`
5. `Mixed`

These challenges reuse the same Level 1-5 map layouts from `Level Mode`, but they remain endless:

- No target score.
- No target progress bar.
- No Level Clear.
- Boundary wrap remains enabled.
- Static walls and moving walls are deadly.
- Portals keep the same teleport behavior and cooldown.
- Normal and big apples use the endless scoring and speed rules.

Pause and Game Over in endless challenge mode include direct paths to restart, return to level select, or return to the main menu.

## Level Mode

`Level Mode` remains target-based. It still uses Level Clear, target scores, the horizontal progress bar, boundary death, static and moving walls, portals, and timed big apples.

The level layouts now also include safe spawn metadata so both Level Mode and Endless Challenges can start the snake away from walls, portals, and moving-wall paths.
