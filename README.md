# Gesture Snake

基于 Python、pygame、OpenCV 和 MediaPipe 的手势控制贪吃蛇游戏。

本版本的核心玩法是：用食指指尖作为目标点，蛇头会平滑追随食指移动。蛇不会瞬间贴到手指位置，而是按当前速度自然靠近目标点，这样可以减少手抖和快速移动造成的断节。

## 主要功能

- 开始菜单：单人模式可玩，闯关模式和双人模式显示“开发中 / Coming Soon”。
- 食指追随控制：食指指尖移动到哪里，蛇头就平滑追向哪里。
- 选项设置：主菜单中可调整手势灵敏度，让小范围手指移动控制全地图。
- 捏合点击：食指指尖和大拇指指尖捏合用于点击菜单按钮。
- 和平手势：Game Over 后主动重开。
- 单手锁定：只锁定第一只进入画面的手，忽略后出现的第二只手。
- 单人无尽模式：右侧游戏区穿墙循环，只有撞到自己身体才会结束。
- 普通苹果和限时红色大苹果：每吃 5 个普通苹果会出现一次 5 秒限时大苹果。
- 轻量增长：普通苹果只让蛇身轻微变长，大苹果增长更明显但不过量。
- 速度成长：吃到的苹果越多，蛇的最大追随速度越快，并带有上限。
- 流畅度优化：使用 dt 时间步、缓存网格背景、连续轨迹采样和平方距离碰撞检测。

## 安装与运行

需要 Python 3.8+ 和摄像头。

安装依赖：

```bash
pip install -r requirements.txt
```

运行游戏：

```bash
python snake.py
```

在当前 Windows 项目环境中，推荐直接使用项目自带虚拟环境：

```bash
.\.venv\Scripts\python.exe snake.py
```

如果你的系统里 `python` 指向 Python 2，请改用 Python 3 解释器运行，例如：

```bash
python3 snake.py
```

## 操作方式

| 操作 | 效果 |
| --- | --- |
| 移动食指指尖 | 蛇头平滑追随食指目标点 |
| 食指和大拇指捏合 | 点击菜单、暂停、结束页面按钮 |
| 主菜单“选项设置” | 调整手势灵敏度 |
| 和平手势 | Game Over 后重新开始 |
| 鼠标点击 | 备用点击方式，方便测试 |
| ESC | 退出游戏 |

只有锁定的手完全丢失超过短暂时间后，游戏才会进入追踪丢失暂停状态；重新识别到手后会自动继续。和平手势不会在游玩中暂停。

## 单人无尽模式规则

1. 蛇在右侧主游戏区移动，左侧摄像头栏不属于游戏区域。
2. 撞到边界不会死亡，会从另一侧穿出。
3. 唯一死亡条件是蛇头撞到自己的身体。
4. 开局有约 3 秒保护时间，避免初始身体重叠误判。
5. 普通苹果一直存在：
   - 得分 +10
   - 蛇身轻微增长
   - 普通苹果重新生成
   - 苹果计数 +1
6. 每吃 5 个普通苹果会生成一个红色大苹果：
   - 只存在 5 秒
   - 得分 +50
   - 蛇身增长更多
   - 与普通苹果不会重叠生成
7. 速度规则：
   - 基础速度：180 像素/秒
   - 每个普通苹果增加：8 像素/秒
   - 最大速度：420 像素/秒

## 手势识别逻辑

MediaPipe Hands 最多检测 2 只手，但单人模式只使用第一只进入画面的手：

- 第一次检测到手时，将它锁定为 active hand。
- 后续出现第二只手时，不会切换控制权。
- 锁定手短暂丢失时，不会马上切换到另一只手。
- 锁定手丢失超过约 1 秒后，才允许重新锁定新的第一只手。

控制数据只来自锁定手：

- 食指指尖：MediaPipe landmark 8
- 大拇指指尖：MediaPipe landmark 4
- 捏合点击：landmark 8 和 4 的归一化距离低于阈值时触发
- 和平手势：食指和中指伸出、无名指和小指收起时触发

摄像头画面的完整区域都可用于手势判定，不再需要把手放进黄色框。

## 手势灵敏度

主菜单中有“选项设置”，可以调整手势灵敏度：

- 标准：食指位置基本按摄像头全画面映射到游戏区。
- 灵敏：手指小一些的移动会映射到更大的游戏范围。
- 高灵敏：默认选项，适合手只在摄像头中间区域活动。
- 超高：很小的手指移动就能覆盖全地图。

灵敏度会围绕摄像头画面中心放大食指位移。例如在高灵敏或超高灵敏度下，手不需要移动到摄像头边缘，也能让蛇追向地图边缘。

## 游戏状态

- MENU：开始菜单，选择单人/闯关/双人。
- OPTIONS：从主菜单进入，调整手势灵敏度。
- PLAYING_SINGLE：单人无尽模式。
- PAUSED：暂停状态。只有手丢失才会触发追踪暂停，重新识别到手后自动继续。
- GAMEOVER：显示最终分数和苹果数量，可按钮重开或和平手势重开。
- COMING_SOON：闯关模式和双人模式的预留页面。

## 代码结构

本轮仍保持单文件入口 `snake.py`，主要结构如下：

```text
snake.py
├─ VisionSystem   摄像头、MediaPipe、单手锁定、食指坐标、捏合和和平手势
├─ Button         菜单按钮，支持食指捏合点击和鼠标点击
├─ Food           普通苹果和限时大苹果数据
├─ VisionResult   每帧手势输出数据
├─ GestureTrigger 手势防抖和冷却
└─ SnakeGame      状态机、游戏规则、绘制和主循环
```

## 常见问题

### 中文显示成方框

程序会优先加载 Windows 中文字体文件，例如微软雅黑、黑体、宋体或 Noto Sans SC。如果仍出现方框，请确认 `C:\Windows\Fonts` 中存在可用中文字体。

### 摄像头打不开

请检查：

- 摄像头是否被其他程序占用。
- 系统是否允许 Python 访问摄像头。
- 笔记本或外接摄像头是否正常连接。

程序会在终端输出清晰的摄像头错误提示。

### 调整摄像头分辨率

在 `snake.py` 顶部修改这些设置即可：

```python
CAMERA_INDEX = 0
CAMERA_RESOLUTION = (640, 480)
CAMERA_FPS = 30
```

如果机器性能足够，可以改成 `(960, 540)`；如果卡顿，可以保持或降低到 `(640, 480)`。

### 蛇身断节

蛇身使用连续轨迹采样。快速移动食指时，程序会在蛇头路径上补充中间身体点，避免出现大块空隙。

### 捏合连续触发

捏合点击带有防抖和冷却。只有从“未捏合”变为“捏合”的瞬间才算一次点击，持续捏合不会连续点击。
